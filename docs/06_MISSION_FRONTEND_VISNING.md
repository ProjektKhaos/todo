# 06 Mission: Frontend-visning

## Mål
Skapa `index.php` som visar aktiva To Do-poster på ett tydligt och snyggt sätt.

## Ska visas
- Rubrik
- Text
- Kategori / underkategori
- Status
- Datum start
- Datum slut
- Tid kvar
- Plats
- Bildförhandsvisning
- Ljudspelare
- Videospelare

## Filvisning
### Bild

```html
<img src="uploads/images/exempel.jpg" alt="Bild för uppgift">
```

### Ljud

```html
<audio controls src="uploads/audio/exempel.mp3"></audio>
```

### Film

```html
<video controls src="uploads/video/exempel.mp4"></video>
```

## Filter, enkel version
Lägg gärna till enkla filter:

- Visa alla aktiva
- Visa efter kategori
- Visa efter status
- Visa försenade
- Visa arkiv

## Resultat
Besökare ska kunna se uppgifter på ett snyggt sätt utan att gå in i admin.
