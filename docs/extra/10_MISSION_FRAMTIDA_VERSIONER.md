# 10 Mission: Framtida versioner

## Idéer för nästa steg

### Version 2
- Sökfunktion.
- Sortering på datum, status och kategori.
- Kalenderläge.
- Påminnelser.
- Export/import av JSON.
- Drag-and-drop upload.
- Taggar.
- Prioritet.

### Version 3
- Byta från JSON till SQLite eller MySQL.
- API-endpoints.
- Mobilvy/PWA.
- Push-notiser.
- Fler användare.
- Rollbaserad admin.

## Viktigt
JSON-versionen ska hållas enkel och portabel. Den ska vara lätt att flytta, kopiera och förstå.

När appen växer kan JSON bli för begränsat. Då är nästa naturliga steg SQLite eller MySQL.
