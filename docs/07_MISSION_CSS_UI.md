# 07 Mission: CSS och layout

## Mål
Skapa en tydlig, modern och mobilvänlig design i `assets/style.css`.

## Stil
- Mörkt eller ljust tema, valfritt.
- Tydliga kort för varje uppgift.
- Statusfärger.
- Mobilanpassad layout.
- Enkel adminlayout med tabeller eller kort.

## CSS-kommentar
CSS-filen ska börja med:

```css
/* style.css - huvudstil för To Do JSON App Ⓐ Style */
/* Uppdaterad: 2026-04-26 14:00 | av: KlⒶssⓔ & Ⓐberg */
```

## Statusklasser
Exempel:

```css
.status-ny {}
.status-pagar {}
.status-vantar {}
.status-klar {}
.status-pausad {}
```

## Resultat
Appen ska kännas enkel, tydlig och inte som ett kalt testscript från stenåldern.
